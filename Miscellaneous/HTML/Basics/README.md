# NOTES ➤
````markdown
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>.....</title>
  </head>
  <body>
    <h1>....</h1>
    <p>.....</p>
  </body>
</html>
````

- HTML tags are insensitive to case.

### b tag vs strong tag ➤

- `<b>` = Only visual bold, no meaning, no TTS (Text to Speech) emphasis  
- `<strong>` = Visual bold + meaningful emphasis, supported by screen readers (TTS)
